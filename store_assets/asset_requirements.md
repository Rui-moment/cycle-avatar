# CycleAvatar Store Asset Requirements

## App Icons

### iOS App Icon Requirements
- **1024x1024**: App Store icon (PNG, no transparency)
- **180x180**: iPhone app icon (@3x)
- **120x120**: iPhone app icon (@2x)
- **167x167**: iPad Pro app icon
- **152x152**: iPad app icon (@2x)
- **76x76**: iPad app icon (@1x)

### Android App Icon Requirements
- **512x512**: Google Play Store icon (PNG)
- **192x192**: xxxhdpi launcher icon
- **144x144**: xxhdpi launcher icon
- **96x96**: xhdpi launcher icon
- **72x72**: hdpi launcher icon
- **48x48**: mdpi launcher icon

### Icon Design Guidelines
- **Style**: Modern, clean, recognizable
- **Colors**: Brand colors (primary: #2196F3, accent: #FF5722)
- **Content**: Stylized avatar or muscle/fitness symbol
- **Background**: Solid color or subtle gradient
- **Text**: Avoid text in icon (except logo if very clear)

## Feature Graphics

### Google Play Feature Graphic
- **Size**: 1024 x 500 pixels
- **Format**: PNG or JPEG
- **Content**: 
  - App logo/name
  - Key visual elements (avatar, muscle groups)
  - Tagline: "Smart Training, Real Growth"
  - No text that duplicates store listing

### App Store Preview Video (Optional)
- **Duration**: 15-30 seconds
- **Resolution**: 1080p minimum
- **Content**:
  - Quick app overview
  - Key features demonstration
  - Avatar growth animation
  - No audio required (subtitles recommended)

## Quality Checklist

### Before Submission
- [ ] All required sizes generated
- [ ] No pixelation or artifacts
- [ ] Consistent branding across all assets
- [ ] Text is readable on all screen sizes
- [ ] Colors match brand guidelines
- [ ] Screenshots show realistic, impressive data
- [ ] All assets optimized for file size
- [ ] Localized versions created
- [ ] Legal compliance (no copyrighted content)
- [ ] Accessibility considerations (contrast ratios)

### Testing
- [ ] View assets on actual devices
- [ ] Test in both light and dark modes
- [ ] Verify readability at small sizes
- [ ] Check color accuracy on different displays
- [ ] Validate file formats and sizes

## Brand Guidelines

### Colors
- **Primary Blue**: #2196F3
- **Accent Orange**: #FF5722
- **Success Green**: #4CAF50
- **Warning Yellow**: #FFC107
- **Error Red**: #F44336
- **Background**: #FAFAFA
- **Surface**: #FFFFFF
- **Text Primary**: #212121
- **Text Secondary**: #757575

### Typography
- **Primary Font**: Roboto (Android), SF Pro (iOS)
- **Headings**: Medium weight, 24-32px
- **Body Text**: Regular weight, 16-18px
- **Captions**: Regular weight, 14px

### Visual Style
- **Design Language**: Material Design 3
- **Corner Radius**: 8px for cards, 4px for buttons
- **Shadows**: Subtle elevation shadows
- **Icons**: Material Design icons or custom outlined style
- **Illustrations**: Flat design with subtle gradients
